# 🚀 Utility Tracker

## 💡 Problem

In daily life, users often need to track activity or calculate distance (for travel, walking, or small tasks).
However:

* Manual calculation is time-consuming
* Existing tools are often complex or overloaded with features
* There is no simple, lightweight solution for quick tracking

---

## ✅ Solution

Utility Tracker provides a **simple and efficient web application** that:

* Tracks user activity in real time
* Calculates distance automatically
* Displays results instantly in a clean interface

This makes tracking fast, easy, and accessible for everyday use.
